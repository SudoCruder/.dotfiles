# .dotfiles

## Setup git
`git_setup.sh`


## Install homebrew
`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`


## Symlink files to appropriate places
`make`


## Install all brew dependencies
`brew bundle`


## Install Rust
`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
